# Краткое описание NgUsers

Простое Angular-приложение для CRUD-операций с пользователями. Взаимодействует с тестовым REST API (JSON Server).Реализованы список пользователей, профиль, форма создания/редактирования, маршрутизация, реактивные формы с валидацией.

## Требования к использованию

Node.js (версия 18 или выше)

## Установка и запуск

1. Клонируйте репозиторий или скачайте проект
git clone <https://github.com/kutuzovakatay-sketch/Test_exmple>
cd ng-Users

2. Установите зависимости
npm i

3. Запустите JSON Server (в первом терминале)
npm run server

4. Запустите Angular приложение (в другом терминале, в папке ng-Users)
npm start
